# DeviceMinder - API Gateway

## Description
<p>
    API Gateway for DeviceMinder app has been made for routing (redirecting requests to specify service).
    It also provides stateless authentication by login (giving username and password) in a way of generating JWT Token.
    After successful authentication, the user passes a token in the header of the request to get access to services.
    Permissions are set by type of user - default or admin.
</p>
